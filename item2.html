Codeigniter Framework Extensie

Januari 2015

Traditionele Codeigniter validatieregels kunnen gegroepeerd worden binnen "subarrays", echter dit creÃ«ert herhaling van de individuele regels bij grote projecten.

Als alternatief voor dit systeem heb ik een library extension geschreven zodat in de configuratie een extra array key "groups" toegevoegd kan worden. Op deze manier kan er dynamisch een groep validatieregels gemaakt worden zonder deze opnieuw te moeten declareren.

Als extra functionaliteit kan er ook een nummer tussen de blokhaken worden toegevoegd per validatieregel om zo te bepalen hoe vaak dit veld voorkomt per formulierpost.

Om deze toevoeging werkend te maken moet de oorspronkelijke validatielibrary worden uitgebreid met de functie "run_group".

Vervolgens kan via deze nieuwe functie de juiste groep worden opgehaald voor validatie.
